# grill-me

Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree.

## What it does

When invoked, the skill conducts an intensive interview about your plan or design:

- Walks down each branch of the design tree systematically
- Resolves dependencies between decisions one-by-one
- Provides recommended answers for each question
- Explores the codebase when a question can be answered by code

## Usage

Use when you want to stress-test a plan, get critical feedback on a design, or simply say "grill me".
